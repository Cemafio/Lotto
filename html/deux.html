Pour y jouer c'est simple:

  1. Vous selectionnez la couleur de votre carte dans les parametres,
  2. apres suffit juste d'appuyez sur "distribuez les cartes".
  3. Cliquez sur le bouton gris a gauche pour faire sortir les chiffres.
  4. Quand une chiffre sur votre carte est sorti,cliquez sur le chiffre pur le marquer.
Pour gagner il faut avoir deux ligne marqué
